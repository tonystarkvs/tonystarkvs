What is REST Assured?

REST Assured is a Java library for testing and validating REST APIs. It provides a fluent DSL (domain-specific language) that makes API tests readable and expressive.

Fluent API BDD Style JSON/XML Support Easy Assertions

The BDD pattern: Given / When / Then

given()          // Set up request (headers, auth, body)
  .header(...)
  .body(...)
.when()          // Make the HTTP request
  .get("/users")
.then()          // Assert the response
  .statusCode(200)
  .body("name", equalTo("John"));

This reads like plain English — making tests self-documenting.

Why TestNG?

  • Groups and priorities for test ordering
  • Data-driven testing with @DataProvider
  • Parallel test execution
  • Rich HTML reports
  • Powerful annotations: @BeforeClass, @AfterMethod, etc.

Your first GET test

import io.restassured.RestAssured;
import org.testng.annotations.BeforeClass;
import org.testng.annotations.Test;
import static io.restassured.RestAssured.*;
import static org.hamcrest.Matchers.*;

public class FirstTest {

  @BeforeClass
  public void setup() {
    RestAssured.baseURI = "https://jsonplaceholder.typicode.com";
  }

  @Test
  public void testGetUser() {
    given()
      .when()
        .get("/users/1")
      .then()
        .statusCode(200)
        .body("name", equalTo("Leanne Graham"))
        .body("email", notNullValue());
  }
}

We use jsonplaceholder.typicode.com — a free public test API. No setup needed!

Hamcrest matchers cheatsheet

// Status & headers
.statusCode(200)
.header("Content-Type", containsString("application/json"))

// Body — simple fields
.body("name", equalTo("John"))
.body("age", greaterThan(18))
.body("email", notNullValue())
.body("deleted", is(false))

// Arrays
.body("items", hasSize(5))
.body("ids", hasItems(1, 2, 3))
.body("tags[0]", equalTo("java"))

// Nested JSON
.body("address.city", equalTo("Mumbai"))
.body("company.name", containsString("Tech"))

// Response time
.time(lessThan(2000L))  // under 2 seconds

Soft assertions with TestNG

import org.testng.asserts.SoftAssert;

@Test
public void testUserDetails() {
  Response res = given()
    .when().get("/users/1");

  SoftAssert sa = new SoftAssert();
  sa.assertEquals(res.statusCode(), 200);
  sa.assertEquals(res.jsonPath().getString("name"), "Leanne Graham");
  sa.assertNotNull(res.jsonPath().getString("email"));

  sa.assertAll(); // Reports ALL failures, not just first
}

Soft assertions collect all failures before stopping — great for validating multiple fields.

Key TestNG annotations

public class ApiTest {

  @BeforeSuite   // Runs once before ALL tests
  public void initSuite() { }

  @BeforeClass   // Runs once before this test class
  public void setupClass() {
    RestAssured.baseURI = "https://api.example.com";
  }

  @BeforeMethod  // Runs before EACH @Test method
  public void beforeEach() { }

  @Test(priority = 1, groups = {"smoke"})
  public void testLogin() { ... }

  @Test(priority = 2, dependsOnMethods = {"testLogin"})
  public void testGetProfile() { ... }

  @AfterMethod   // Runs after EACH @Test
  public void afterEach() { }

  @AfterClass    // Runs once after this class
  public void teardownClass() { }
}

Data-driven testing with @DataProvider

@DataProvider(name = "userIds")
public Object[][] provideUserIds() {
  return new Object[][] {
    { 1, "Leanne Graham" },
    { 2, "Ervin Howell" },
    { 3, "Clementine Bauch" }
  };
}

@Test(dataProvider = "userIds")
public void testMultipleUsers(int userId, String expectedName) {
  given()
    .when()
      .get("/users/" + userId)
    .then()
      .statusCode(200)
      .body("name", equalTo(expectedName));
}

Request specification (reusable config)

// In BaseTest.java
public class BaseTest {

  protected RequestSpecification reqSpec;

  @BeforeClass
  public void buildSpec() {
    reqSpec = new RequestSpecBuilder()
      .setBaseUri("https://api.example.com")
      .setContentType(ContentType.JSON)
      .addHeader("Authorization", "Bearer " + getToken())
      .addFilter(new RequestLoggingFilter())
      .addFilter(new ResponseLoggingFilter())
      .build();
  }
}

// In test classes
@Test
public void testEndpoint() {
  given(reqSpec)          // reuse the spec!
    .when().get("/users")
    .then().statusCode(200);
}

Authentication

// Basic Auth
given().auth().basic("user", "pass")

// Bearer Token
given().header("Authorization", "Bearer " + token)

// OAuth2
given().auth().oauth2(accessToken)

// API Key
given().queryParam("api_key", "abc123")
// or
given().header("X-API-Key", "abc123")

Logging for debugging

// Log everything
given().log().all()
  .when().get("/users")
  .then().log().all();

// Log only on failure
given()
  .when().get("/users")
  .then()
    .log().ifValidationFails()
    .statusCode(200);
